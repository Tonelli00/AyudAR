# AyudAR - Aplicación para promover la colaboración social
**AyudAR** es una plataforma web orientada a conectar personas, organizaciones no gubernamentales (ONG) y empresas con el objetivo de facilitar la colaboración en causas sociales.

La aplicación permite que quienes necesitan ayuda puedan encontrar personas dispuestas a colaborar, ya sea mediante:
- Donaciones económicas.
- Voluntariado.
- Aportes profesionales.
- Difusión de causas.

La idea nace a partir de una problemática común: muchas personas tienen la intención de ayudar, pero no encuentran fácilmente organizaciones confiables, causas acordes a sus intereses o formas de colaborar según sus habilidades y disponibilidad.

Actualmente, gran parte de las causas benéficas se difunden a través de redes sociales tradicionales, donde las publicaciones tienen una vida útil muy corta y suelen perderse entre una gran cantidad de contenido. Esto dificulta que las personas puedan descubrir oportunidades de colaboración y que las organizaciones logren alcanzar a potenciales voluntarios o donantes.

AyudAR busca centralizar estas causas en una plataforma dedicada exclusivamente al impacto social, permitiendo que las organizaciones puedan visibilizar sus necesidades y que los usuarios encuentren fácilmente formas concretas de aportar.

---

# Objetivo Principal
El objetivo principal de **AyudAR** es crear un espacio digital que promueva la colaboración y el crecimiento social, conectando personas, organizaciones y comunidades para facilitar acciones solidarias y generar un impacto positivo en la sociedad.

# Funcionalidad principales
## Registro/Creación de nuevos usuarios
- Registro e inicio de sesión 
- Creación de perfil personal
- Historial de donaciones realizadas
## Organizaciones / Usuarios
- Perfil para ONG 
- Creación de postulaciones
- Busqueda de voluntarios (ONGs)
- Publicaciones para causas beneficas
- Control en tiempo real de lo recaudado o voluntarios postulados
## Voluntariado
- Postulaciones a propuestas / Proyectos
## Comunicación
- Sistema de mensajeria para comunicación entre usuarios
